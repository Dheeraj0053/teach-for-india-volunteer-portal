# Home page images

Add your images here, then refresh the site.

| File | Used for | Recommended size |
|------|----------|------------------|
| `hero.jpg` | Full-width hero background | 1920×900 or wider |
| `hero.webp` | Optional (faster load) | Same as above |

**Or** set a URL in `.env`:

```
VITE_HERO_IMAGE=https://your-cdn.com/your-photo.jpg
```

If `hero.jpg` is missing, a temporary stock photo is used until you add your own.
